## Screenshot Guidelines

### Design Principles
- **Modern & Clean**: Minimalist design with clear visual hierarchy
- **Dark Theme**: Professional dark theme for security operations
- **Color Coding**: Consistent color scheme (green=safe, yellow=warning, red=critical)
- **Real-Time Updates**: Visual indicators for live data streaming
- **Responsive**: Works on desktop, tablet, and mobile

### Color Palette
- **Primary**: #0066FF (Blue) - Trust, security
- **Success**: #00C853 (Green) - Safe, compliant
- **Warning**: #FFB300 (Yellow) - Caution, review needed
- **Critical**: #E53935 (Red) - Threat, violation
- **Background**: #1E1E1E (Dark gray) - Professional, easy on eyes
- **Text**: #FFFFFF (White) - High contrast

### Typography
- **Headers**: Inter, 24-32px, Bold
- **Body**: Inter, 14-16px, Regular
- **Metrics**: Inter, 18-24px, Semi-Bold
- **Code**: JetBrains Mono, 13px, Regular

### Key UI Components
- **Cards**: Rounded corners (8px), subtle shadows
- **Charts**: Interactive with hover tooltips
- **Tables**: Sortable, filterable, paginated
- **Forms**: Clear labels, inline validation
- **Alerts**: Toast notifications, banner alerts
- **Modals**: Centered, backdrop blur
